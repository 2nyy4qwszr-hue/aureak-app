# QA Gate 1 — Story 51-7 : Sidebar collapse smooth animation
Date : 2026-04-05
Agent : Developer Agent BMAD (Amelia)
Status : PASS

## Design Critic — Animation fluide

### AC1 — Animation 280ms ease largeur
- PASS : `transition: 'width 0.28s ease'` ajouté dans le style inline de la YStack sidebar
- `overflow: 'hidden'` ajouté pour éviter tout débordement pendant la transition
- La prop `width={sidebarWidth}` (52 ou 220) est interpolée automatiquement par le browser CSS

### AC2 — Masquage animé des labels (collapse)
- PASS : `labelsVisible` state séparé de `sidebarCollapsed`
- Au collapse : `setLabelsVisible(false)` immédiatement, `setSidebarCollapsed(true)` après 80ms
- Tous les labels ont `transition: 'opacity 0.1s ease'` — disparaissent avant la réduction de largeur
- Couvre : labels AUREAK/Administration, labels groupes nav, labels items nav, hints raccourcis, separators, GlobalSearch, NotificationBadge, admin user pill

### AC3 — Apparition animée des labels (expansion)
- PASS : À l'expansion, `setSidebarCollapsed(false)` immédiat, `setLabelsVisible(true)` après 180ms
- Labels apparaissent seulement quand la largeur est quasi-finale → pas de débordement / clignotement

### AC4 — Tooltip au hover en mode collapsed
- PASS : `NavTooltip` composant créé dans `aureak/apps/web/components/NavTooltip.tsx`
- `HoverablePressable` wrapper typé (interface étend PressableProps) pour `onMouseEnter/Leave`
- `hoveredHref` state → `visible={sidebarCollapsed && isHovered}` sur chaque NavTooltip
- Tooltip invisible en mode expanded (`sidebarCollapsed=false`)

### AC5 — Tooltip design cohérent
- PASS : `backgroundColor: colors.background.elevated`, `borderRadius: radius.xs` (6), `paddingHorizontal: 8`, `paddingVertical: 4`
- `Text` en `colors.text.primary`, `fontSize: 12`
- `position: 'absolute'`, `left: 36`, `transform: [{ translateY: -14 }]` (centrage vertical)
- `zIndex: 100`, `pointerEvents: 'none'`, `whiteSpace: 'nowrap'`

### AC7 — Bouton toggle animé rotation
- PASS : `Animated.Value` initialisé depuis localStorage (0 = expanded, 1 = collapsed)
- `Animated.timing(toggleRotation, { toValue, duration: 280, useNativeDriver: true })`
- `rotateInterp = interpolate([0,1] → ['0deg','180deg'])` appliqué via `Animated.View`
- Synchronisé avec l'animation de largeur 280ms

### AC8 — Zéro layout shift contenu principal
- PASS : Zone contenu `flex: 1` — suit naturellement le flex container CSS lors de la transition sidebar

## Verdict Design Critic
- Tous les ACs visuels : PASS
- Animation fluide premium sans jump instantané
- Labels ne débordent pas pendant la transition
- Tooltips discrets et cohérents avec le design system
