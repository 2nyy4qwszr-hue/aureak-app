- `StreakTile` affiche l'état vide avec message AC2 : "Aucune streak active" + sous-texte
- RESULT: **PASS**

### EG2 — Égalité de score (deux joueurs avec même streak)
- Sort: `b.streak - a.streak || a.childId.localeCompare(b.childId)`
- Résultat déterministe : en cas d'égalité, trié par childId lexicographique (UUID stable)
- RESULT: **PASS**

### EG3 — Joueur sans profil dans `profiles`
- `nameMap.get(t.childId) ?? 'Joueur'` — fallback "Joueur" si profil absent
- `InitialsAvatar` avec "Joueur" → initiales "JO" (2 lettres) — OK
- RESULT: **PASS**

### EG4 — Erreur API attendance_records

- `StreakTile` reçoit `players=[]` (via `data ?? []`) → affiche empty state sans crash
- RESULT: **PASS**

### EG5 — Nom vide ou 1 mot
- `name.split(' ').map(w => w[0] ?? '').join('').slice(0, 2).toUpperCase()` — opérateur `??` empêche crash sur `undefined` si mot vide
- Nom 1 mot (ex: "Luca") → initiales "L" — acceptable
- RESULT: **PASS**
