## Bug Hunter — Points inspectés

**[PASS] Conflit ⌘K** : L'import de `useCommandPalette` dans `useKeyboardShortcuts` a été retiré (créait une instance séparée avec `isOpen=false` permanent). L'isolation est gérée par `isInputFocused()` — le TextInput de CommandPalette est focusé quand la palette est ouverte, ce qui bloque naturellement les chords.

**[PASS] Typing dans input** : Guard `isInputFocused()` en première ligne du handler — tag `INPUT|TEXTAREA|SELECT` + `isContentEditable`.

**[PASS] Cleanup listeners** : `document.removeEventListener` dans le return du `useEffect` + `clearChordTimer()` au démontage via second `useEffect`.

**[PASS] Timer leak** : `chordTimerRef` nettoyé dans `clearChordTimer()` appelé à 3 endroits : Phase 1 (reset avant nouveau timer), Phase 2 succès, cleanup démontage.

**[PASS] Modificateurs** : Les touches avec `ctrlKey|metaKey|altKey` resetent le préfixe (évite d'activer un chord sur Ctrl+G par exemple).

**[PASS] Double-préfixe** : `PREFIX_KEYS.has(key)` uniquement si `!prefixKey` — impossible d'empiler G+G.
