## Bugs détectés (QA Hunter)

### Flicker au chargement (FOUC)
- Analyse : pas de SSR (Expo Router web = SPA client-side). La lecture localStorage est synchrone dans `useState(() => getInitialTheme())`. Le thème est résolu AVANT le premier render React → pas de FOUC.
- Verdict : **aucun flicker détecté** par analyse statique.

### prefers-color-scheme sync
- Un `MediaQueryList.addEventListener('change', ...)` est enregistré dans `useEffect`.
- Condition de guard : si `localStorage` contient une valeur explicite (`'light'`/`'dark'`), le changement OS est ignoré. Si localStorage vide/corrompu, le changement OS est appliqué.
- Verdict : **comportement correct**.

### localStorage corrompu
- `getInitialTheme()` : validation stricte `if (saved === 'light' || saved === 'dark')` → toute autre valeur (null, undefined, valeur inconnue) est ignorée et passe au fallback `prefers-color-scheme`.
- `useEffect` persistence : entouré de try/catch avec console guard.
- Verdict : **robuste**.
