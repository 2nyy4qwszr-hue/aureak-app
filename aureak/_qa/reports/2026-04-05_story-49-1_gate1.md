## 2. Findings

### [BLOCKER] B1 — `setClubsLoading(false)` sans finally (new.tsx:99-102)

**Fichier** : `aureak/apps/web/app/(admin)/users/new.tsx:99-102`

**Problème** : `setClubsLoading(false)` était placé uniquement dans le `.then()` de `listClubDirectory`. Si la promesse est rejetée (erreur réseau, timeout, erreur Supabase), `setClubsLoading` reste à `true` indéfiniment — le spinner "Chargement..." bloque l'affichage du sélecteur de club pour toute la session.

**Règle violée** : Try/finally obligatoire sur tout state setter de chargement/sauvegarde (CLAUDE.md).

**Correction appliquée** :
```typescript
listClubDirectory({ actif: true, pageSize: 200 })
  .then(({ data }) => {
    setClubs((data ?? []).map(c => ({ id: c.id, nom: c.nom })))
  })
  .catch((err: unknown) => {
    if (process.env.NODE_ENV !== 'production') console.error('[users/new] listClubDirectory error:', err)
  })
  .finally(() => {
    setClubsLoading(false)
  })
```

**Statut** : CORRIGÉ
